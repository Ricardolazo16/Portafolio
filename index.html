MisProyectos

Sistema de Gestión de Biblioteca

Desarrollé un sistema de gestión de biblioteca como parte de un proyecto académico. La aplicación permite a los usuarios buscar y solicitar libros, así como llevar un registro de los préstamos y devoluciones. Incluye funciones de administración para el personal de la biblioteca.

Plataforma de Aprendizaje en Línea

Desarrollé una plataforma de aprendizaje en línea que ofrece cursos interactivos y contenido multimedia. La plataforma incluye características como inscripción a cursos, seguimiento del progreso del estudiante y foros de discusión. Además, integré un sistema de recomendación basado en algoritmos de filtrado colaborativo para sugerir cursos personalizados a los usuarios.

Aplicación de Rastreo de Hábitos

Desarrollé una aplicación móvil para el seguimiento de hábitos personales. Los usuarios pueden establecer metas diarias, recibir recordatorios y realizar un seguimiento de su progreso a lo largo del tiempo. Incluye gráficos visuales para mostrar el rendimiento.

Identificación de Rostros

Desarrollé un sistema de identificación de rostros utilizando técnicas avanzadas de Machine Learning. El proyecto se centró en el reconocimiento facial en tiempo real, utilizando una combinación de algoritmos de extracción de características y redes neuronales convolucionales (CNN). Implementé la detección de rostros en imágenes y posteriormente entrené un modelo para reconocer individuos específicos. La aplicación tiene aplicaciones prácticas en la seguridad y el acceso biométrico.
